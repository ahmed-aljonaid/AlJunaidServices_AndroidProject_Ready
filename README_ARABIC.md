# بوابة الخدمات الطلابية والأكاديمية - AlJunaid Services Android Project

مرحباً بك! تم توليد كود هذا المشروع المتكامل بلغة **Kotlin** وباستخدام واجهة **Jetpack Compose** الحديثة ودعم كامل لـ **RTL** والاتجاه من اليمين إلى اليسار تلقائياً.

## 📁 هيكلية الملفات المضمنة في الملف المضغوط:
1. **MainActivity.kt**: النشاط الرئيسي الحاوي لهيكل التطبيق الأساسي والتنقل.
2. **AppScreens.kt**: شاشات التطبيق الأساسية (الرئيسية، الخدمات، معرض الأعمال، الآراء، اتصل بنا، لوحة التحكم).
3. **MainViewModel.kt**: الـ ViewModel لإدارة الحالات والاتصال بالإنترنت مع نظام Fallback الاحتياطي الفوري.
4. **SheetsApiService.kt**: منفذ اتصال Retrofit وجداول البيانات.
5. **Theme.kt**: سمة التطبيق الأساسية وألوان الوضع المظلم وتعيين عائلة خط "Tajawal" برمجياً.
6. **build.gradle.kts**: ملف بناء الاعتمادات والاتصالات ومكاتب التشغيل (Retrofit, Coil, Media3/ExoPlayer).
7. **GoogleAppsScript.js**: كود الـ Apps Script لاستقبال طلبات الـ Get والجلب والإضافة والموافقة.
8. **Index.html**: لوحة تحكم ويب إضافية لإدارة ومراجعة التقييمات وقبولها.
9. **.github/workflows/android.yml**: كود البناء السحابي التلقائي لاستخراج ملف APK من GitHub مباشرة ومجاناً!

## 📲 كيفية الحصول على ملف APK وتثبيته على الجوال:

### الطريقة الأولى (الموصى بها - بضغطة زر عبر أندرويد ستوديو):
1. قم بفك الضغط عن هذا الملف على كمبيوترك.
2. افتح برنامج **Android Studio** واختر **Open Project** ثم حدد المجلد المستخرج.
3. دع البرنامج يقوم بعملية التثبيت والمزامنة التلقائية لـ **Gradle Sync**.
4. من القائمة العلوية للبرنامج، اختر: **Build** -> **Build Bundle(s) / APK(s)** -> **Build APK(s)**.
5. خلال ثوانٍ، ستظهر لك نافذة صغيرة في أسفل يمين الشاشة، انقر على كلمة **locate** لتفتح لك المجلد الحاوي على ملف الـ **app-debug.apk** مباشرة! انقله إلى هاتفك وثبته واستمتع.

### الطريقة الثانية (سحابية بالكامل - دون برامج على جهازك):
1. قم بإنشاء مستودع (Repository) جديد على حسابك في موقع GitHub.
2. ارفع كود المشروع المستخرج بالكامل إليه.
3. سيقوم ملف الـ GitHub Action الذكي المرفق بتشغيل بناء سحابي فوري ومجاني للتطبيق!
4. توجه لتبويب **Actions** في حسابك بـ GitHub، وستجد ملف الـ **APK** جاهزاً للتحميل في غضون دقيقة واحدة فقط!

## 🚀 طريقة ربط لوحة التحكم وجداول البيانات:
1. اربط تطبيقك بـ Google Sheets عبر نشر ملف `GoogleAppsScript.js` كويب هوك (Web App).
2. انسخ الرابط الناتج من النشر والصقه في الـ Base URL داخل ملف `MainViewModel.kt`.
3. استمتع بتحديث محتوى التطبيق بالكامل مباشرة من جهازك وجوالك!
